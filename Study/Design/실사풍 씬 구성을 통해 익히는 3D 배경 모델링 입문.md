# 1강.
> ## 게임에서 배경작업이 하는 역할
>> 배경: 배경원화가, 배경모델러, 배경레벨러
>>etc)  캐릭터, 이펙트, 애니메이션
>>배경 작업자
>>- 배경원화가: 컨셉전달
>>- 배경모델러: 프랍제작
>>- 배경레벨러: 레벨, 라이팅
>>- 테크니컬 아트디렉터
>
>## 실사와 스타일라이즈
>>- 실사
>>- 반실사
>>- 캐주얼
>>- 스타일라이즈
>> 양식화 하는 이유: 심미성과 게임성(색분할로 구분이 쉽게 하기 위하여)
> ## 작업 파이프라인, 소프트웨어 설명
>> 1. 더미작업
>> 	화이트 메쉬(더미)로 배치하며 게임의 전체 무드와 스케일링을 확인함
>> 2. 엔진 가배치
>> 3. 어셋 실제작(모델링-지브러쉬-텍스쳐링)
> >4. 엔진 레벨링 및 라이팅 작업
> >	순서: 3ds max(모델링) -> Hard Surface ZBrush(스컬핑) -> Substance Painter(텍스처링) -> Unreal Engine(레벨링과 라이팅)
