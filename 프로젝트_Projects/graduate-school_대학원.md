# 대학원

이 노트는 [[User Dashboard]]의 빠른 이동 링크이자, 대학원 지식 운영의 프로젝트 기준점이다.

## 목적

대학원 관련 자료를 단순히 많이 쌓지 않고, 나중에 다시 검색하고 재사용할 수 있게 남긴다. 강의노트, 논문 후보, 연구 아이디어, 출처 조사, literature map은 서로 다른 상태로 관리한다.

## 제리에팀 역할

- 제리에 / `serie`: 최종 보존 기준, Obsidian 구조, retrieval rule, 장기 연구 지도.
- 메토데 / `methode`: 읽기 쉬운 요약, 태그·링크 후보, 보존 후보 packet.
- 젠제 / `sense`: source quality, retrieval 가능성, 보존 합격/보류/탈락 검수.
- 레르넨 / `lernen`: literature note template, BibTeX/Markdown 정리, 백업·이관 절차.
- 게나우 / `genau`: citation mismatch, 검색 실패, routing incident 조사.
- 팔슈 / `falsch`: 논문 검색 false-positive screening.

## 자료 상태

- `candidate`: 검색되었지만 아직 원문·관련성 검토 전.
- `screened`: 팔슈 또는 메타데이터 검토를 통과한 후보.
- `summarized`: 메토데가 읽기 쉬운 요약을 만든 상태.
- `quality-checked`: 젠제가 보존 가능성을 판정한 상태.
- `accepted`: 제리에가 장기 서가 반영을 허용한 상태.
- `rejected`: 관련성 부족, 출처 약함, 중복, 안전 위험 등으로 보존하지 않는 상태.

## 노트 최소 필드

대학원 관련 장기 노트 후보는 최소한 다음을 가져야 한다.

- source title
- source URL / DOI / arXiv ID / 문서 경로
- read status
- source level: metadata / abstract / full text / lecture note / conversation
- summary owner
- quality examiner
- confirmed facts
- open questions
- reuse condition
- accepted by Serie 여부

- 실제 내용은 하위 노트와 자료 폴더에 쌓는다.
- 이 프로젝트 노트는 이동·삭제 없이 탐색 기준점으로 유지한다.
- 실제 vault 이동, 삭제, 대량 태깅, 대량 rename은 힘멜 승인 후에만 한다.
