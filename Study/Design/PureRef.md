## ✅ **전체 사진 & Canvas 정렬 / 정돈**

|**기능**|**단축키 (Windows/Linux)**|
|---|---|
|전체 이미지 선택|`Ctrl + A`|
|최적 정렬 (Arrange Optimal)|`Ctrl + P`|
|모든 이미지 패킹/정리|`Ctrl + Alt + P`|
|추가 순서로 정렬|`Ctrl + Alt + A`|
|이름 순으로 정렬|`Ctrl + Alt + N`|
|경로 순 정렬|`Ctrl + Alt + D`|
|순서 정렬|`Ctrl + Alt + O`|
|위쪽 정렬|`Ctrl + Up`|
|아래쪽 정렬|`Ctrl + Down`|
|왼쪽 정렬|`Ctrl + Left`|
|오른쪽 정렬|`Ctrl + Right`|
|가로 균등 배치|`Ctrl + Alt + Shift + Up`|
|세로 균등 배치|`Ctrl + Alt + Shift + Down`|
|캔버스 자동 맞춤|`Ctrl + O`|
|Pack & Optimize|`Ctrl + Alt + P`|
|||

## 🖼️ **이미지 개별 조작**

|**기능**|**단축키 / 조작 방식**|
|---|---|
|이동|**왼클릭 + 드래그**|
|확대/축소|**Ctrl + Alt + 드래그**|
|반전 (좌우/상하)|**Alt + Shift + 드래그**|
|회전|**Ctrl + 드래그**|
|45° 스냅 회전|**Ctrl + Shift + 드래그**|
|자르기 (Crop)|**C + 마우스 드래그**|
|자른 영역 이동|**V + 드래그**|
|자르기 초기화|`Ctrl + Shift + C`|
|변형(Scaling/Rotation) 초기화|`Ctrl + Shift + T`|
|이미지 앞으로 보내기|`Up`|
|이미지 뒤로 보내기|`Down`|
|이미지 색상코드 보기|**Left MB + S**|
|이미지 좌표 보기|**Left MB + D**|
|불투명도 변경|**Ctrl + Shift + Alt + Drag**|
|선택된 이미지만 확대(포커스)|`Space`|
|캔버스 전체 포커스|`Ctrl + Space`|
|||
